## Get started server(app.js)

Install the dependencies...

```bash 
npm i express
npm i nodemon
```

...then start server(https://localhost:7000):

```bash
nodemon app.js
```

## Get started App.svelte

Install the dependencies...

```bash
npm install
```

...then start [Rollup](https://localhost:5000):

```bash
npm run dev
```
